# Gauge/Calibration Management指揮台

管理廠內量測儀器與量具之校驗週期、履歷與合格狀態，確保檢驗數據有效性。

- 原始系統：Gauge/Calibration Management（量具校驗管理）
- 產業分類：智慧製造｜品質管理
- 使用部門：計量室、品保部、生產部、設備工程部
- 工作流程：量具建檔→校驗排程→送校/內校執行→結果登錄→合格標示發放
- AI 應用：校驗週期智慧調整、異常量具風險預警
- 主要 KPI：校驗準時率、量具逾期比例、校驗成本
- 代表產品：Blue Mountain RAM、GAGEtrak、Beamex CMX、Fluke MET/CAL
